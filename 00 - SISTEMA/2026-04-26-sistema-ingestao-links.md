# Sistema de Ingestão de Links

## 🧠 Essência

Qualquer URL enviada é automaticamente lida, resumida e transformada em nota estruturada no vault — com aplicação prática ao negócio, versionada no Git.

## ⚙️ Aplicação

* Enviar qualquer link (artigo, vídeo, referência, estratégia) para capturar o conhecimento sem perder tempo com resumos manuais
* Conteúdo capturado já conectado ao sistema central
